# ecommerce-dashboard

## Deskripsi
Dashboard interaktif untuk menganalisis distribusi pelanggan e-commerce Brasil berdasarkan negara bagian, wilayah, dan kota.


## Setup
### 1. Clone repository
```bash
git clone https://github.com/elviyanti/ecommerce-dashboard.git
cd ecommerce-dashboard
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Jalankan dashboard
```bash
cd dashboard
streamlit run dashboard.py
```

### 4. Buka di browser
Setelah menjalankan perintah di atas, buka browser dan akses:
```
http://localhost:8501
```
